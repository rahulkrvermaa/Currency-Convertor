# 💱 Currency Converter

A simple and fast **Currency Converter web app** built using **React + Vite**.
It allows users to convert between different currencies with real-time data.

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/rahulkrvermaa/Currency-Convertor.git
```

2. Navigate to project folder:

```bash
cd Currency-Convertor
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm run dev
```
